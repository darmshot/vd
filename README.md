### Simple Git Flow util


Available commands:


```
hs - Hotfix Start
hf - Hotfix Finish
```


```
rs - Release Start
rs --major - Release Start and upgrade major version
rs --first - Release Start first
rf - Release Finish
rf --major - Release Finish and upgrade major version
rf --first - Release Finish first
```


```
fs - Feature Start
ff - Feature Finish
```
